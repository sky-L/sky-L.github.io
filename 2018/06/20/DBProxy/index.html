QA

  • Q : 报错 proxy Warning - Syntax Forbidden Prepare 的错误怎么解决?

    设置 PDO 的的处理方式，PDO::ATTR_EMULATE_PREPARES => true 解决。

  • Q 报错 chart_set 等错误？

    不支持自定义字符集，默认使用数据库配置的。
